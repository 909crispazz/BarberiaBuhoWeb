# BarberiaWeb - Sistema de Gestión de Citas

🌐 **[Ver el proyecto en vivo](https://exquisite-heliotrope-5b8187.netlify.app)**
🌐 **[Ver el proyecto en vivo](https://exquisite-heliotrope-5b8187.netlify.app/admin)**

Una aplicación web (PWA) para la gestión integral de citas de una barbería. Cuenta con una interfaz para clientes enfocada en la conversión y un panel administrativo seguro para el control total de la agenda diaria del barbero.

## 🚀 Características Principales

* **Portal de Clientes (`index.html`):** Interfaz de agendamiento. Incluye un flujo de validación donde el cliente debe depositar el 50% del servicio y enviar el comprobante directamente por WhatsApp para asegurar su espacio.
* **Panel de Administración (`admin.html`):** Dashboard privado que permite revisar citas pendientes, controlar los horarios de forma dinámica, visualizar la agenda por fecha y registrar citas manualmente.
* **Gestión de Aprobaciones:** Integración en el panel para aprobar citas pendientes con un enlace directo a WhatsApp para interactuar con el cliente.
* **Autenticación (`login.html`):** Sistema de inicio de sesión para proteger el acceso al panel administrativo.

## 🛠️ Tecnologías Utilizadas

* **Frontend:** HTML5, CSS3 (`style.css`), y JavaScript Vanilla (ES6 Modules).
* **Backend y Base de Datos:** Firebase Firestore (almacenamiento en tiempo real de la agenda) y Firebase Auth (seguridad y control de acceso).
* **Despliegue:** Netlify.
